# Evergreen Insurance — Quote Page

A small static web app used across the FSE Leaders Phase 1 program. Each week,
you'll open this app (or a week-specific variant of it) and make a small,
scoped change that illustrates what your engineering teams actually do.

## What's in this folder

- `index.html` — the page structure (HTML)
- `styles.css` — the visual styling (CSS)
- `app.js` — the behavior, including the premium calculation (JavaScript)

## How to open it

1. Open this folder in Visual Studio Code.
2. Double-click `index.html` to open it in your default browser — or right-click
   `index.html` in VS Code and choose **Open with Live Server** (if the Live
   Server extension is installed).
3. Fill in the form and click **Calculate Premium** to see it in action.

## What Week 1 focuses on

This week is about developer tools — the command line, Git, and VS Code.
You are **not** asked to write any code this week. You'll just open the
folder, view the file tree, run the app, make a one-line change, and see
how VS Code surfaces that change in its Source Control panel.

> **Why this matters:** Before an engineer writes a line of real code in
> a given day, they open a folder like this, run it to confirm it works,
> and check Git status to see what's changed. It's the muscle memory of
> professional development — and seeing it once gives you context for
> every conversation with your engineers afterward.

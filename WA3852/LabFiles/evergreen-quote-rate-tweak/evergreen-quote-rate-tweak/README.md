# Evergreen Quote — Rate Tweak (starter)

This is the Week 5 starter for the Leaders track. It is the full
Evergreen Insurance Quote Page you have built up in Weeks 2–4 —
HTML + CSS + a small `app.js` that calculates an estimated premium.

Follow `evergreen-quote-rate-tweak.adoc` in the course materials.
The lab walks you through reading `app.js`, setting a breakpoint in
Chrome DevTools, stepping through the paused function, and adjusting
one value in the `ratePerThousand` lookup table.

## Files

- `index.html` — page structure (header, hero, benefits, quote form, footer)
- `styles.css` — styling inherited from Week 4 (CSS variables, Flexbox, media query)
- `app.js` — slider handler + `calculatePremium` function + form submit wiring

## Run it

Open the folder in VS Code, then click **Go Live** in the status bar.
The page loads at `http://127.0.0.1:5500/`.

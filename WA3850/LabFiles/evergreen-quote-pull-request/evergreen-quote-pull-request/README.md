# Evergreen Insurance Quote Page

Small static web project used throughout the FSE Leaders Phase 1
program. The page has three files:

- `index.html` — structure (header, hero, benefits row, quote form, footer)
- `styles.css` — layout and colors (CSS variables, Flexbox, one responsive media query)
- `app.js` — behavior (slider update, premium calculation, form submit handler)

Open `index.html` with VS Code's *Live Server* extension and the page
runs at `http://127.0.0.1:5500`.

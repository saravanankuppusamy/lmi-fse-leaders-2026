# Evergreen Quote — Responsive Layout

This is the Week 4 starter for Liberty Mutual FSE Leaders (Phase 1).

## What Week 4 focuses on

This week you extend the Evergreen Quote Page with Flexbox and a media
query. The starter carries forward the Week 3 stylesheet (CSS variables
and the Liberty Mutual blue rebrand) and adds a new **"Why Evergreen"**
benefits section that currently stacks vertically. Your job is to:

1. Turn the three benefit cards into a single Flexbox row that sits
   neatly across the page at desktop width.
2. Add a media query that flips the row back to a vertical stack on
   narrow screens (phones).

You will use Chrome DevTools extensively — the visual `justify-content`
and `align-items` pickers are the fastest way to see Flexbox behavior
change in real time.

## Files

- `index.html` — the page structure, now with a `<section class="benefits">` block
- `styles.css` — the Week 3 stylesheet plus placeholder rules for the new benefits section
- `app.js` — unchanged from Week 3 (quote form logic)

## How to run

1. Open this folder in VS Code.
2. Right-click `index.html` and choose **Open with Live Server** (or click
   "Go Live" in the VS Code status bar).
3. The page should open in your browser at `http://127.0.0.1:5500/`.
4. Open Chrome DevTools (F12 or Right-click → Inspect).

Full step-by-step lab instructions are in the course material under
`evergreen-quote-responsive-layout.adoc`.
